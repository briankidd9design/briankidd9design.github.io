Software Test Development Engineer

The Software Test Development Engineer course is divided into seven main topics. The first lessons cover black box testing otherwise know as User Interface Testing. This course is very popular as it has several tasks that both the Programmer and Program Manager deal with such as scheduling milestones for both testing and the developer.

Lesson Topics:

  • Lesson 1: Testing Basics
  • Lesson 2: Test Case Development
  • Lesson 3: Testing Reports
  • Lesson 4: Test Plan Overview
  • Lesson 5: Systems Testing
  • Lesson 6: Web Testing
  • Lesson 7: Internet Browsers

Requirements:

  • Know how to program in Basic Python or another language like C, C++ or C#.
  • The following duties are shared responsibilities with the Tester and Program Manager
  • Dealing with Development and Testing Teams scheduling
  • Working with OEM (Original Equipment Manufacturer)
  • Scheduling for product ship date
  • Responsible for ensuring all goals and deadlines are met
  • Setup software to test
